## How to use
1. clone the repo
```
git clone https://github.com/rapirent/vimrc.git
```

2. copy the `_vimrc` as the `.vimrc` to the $HOME directory

3. execute the vim, and enter the command
```
vim
:PlugClean
:PlugInstall
:PlugUpdate
:q!
```
