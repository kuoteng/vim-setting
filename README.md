# vimrc

## Summary
- 這是我的vimrc，最後修改時間:12/10, 2016

## How to
1. clone the repo
```
git clone https://github.com/rapirent/vimrc.git
```
2. execute the script
```
cd vimrc
sh ./install.sh
```
3. execute the vim, and enter the command
```
vim
:PlugInstall
```
4. quit the vim by command
```
:q!
```
